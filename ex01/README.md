# Exercício 1 - Número Positivo

## 📌 Descrição
O programa solicita ao usuário que digite um número positivo.  
Caso o valor informado seja negativo, o sistema continuará pedindo até que um número válido seja digitado.

## 🛠️ Tecnologias
- JavaScript

## ▶️ Execução
Executar o arquivo em um navegador.

## 💡 Lógica utilizada
- Estrutura de repetição `while`
- Validação de entrada de dados

## ✅ Resultado esperado
O programa só aceita números positivos e exibe o valor válido ao final.
