# Exercício 2 - Média de Notas

## 📌 Descrição
O programa permite que o usuário digite várias notas entre 0 e 10.  
A entrada de dados é encerrada quando um número negativo é informado.  
Ao final, o sistema calcula e exibe a média das notas válidas.

## 🛠️ Tecnologias
- JavaScript

## ▶️ Execução
Executar o arquivo em um navegador.

## 💡 Lógica utilizada
- Estrutura de repetição `while`
- Validação de intervalo (0 a 10)
- Acumulador e contador

## ✅ Resultado esperado
Exibir a média das notas digitadas corretamente.
