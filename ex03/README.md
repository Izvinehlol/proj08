# Exercício 3 - Total de Compras

## 📌 Descrição
O programa solicita valores de compras ao usuário.  
A soma é realizada continuamente até que o valor 0 seja digitado.  
Ao final, o total da compra é exibido.

## 🛠️ Tecnologias
- JavaScript

## ▶️ Execução
Executar o arquivo em um navegador.

## 💡 Lógica utilizada
- Estrutura de repetição `while`
- Acumulador de valores

## ✅ Resultado esperado
Exibir o valor total das compras.
