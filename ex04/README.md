# Exercício 4 - Nome Completo

## 📌 Descrição
O programa solicita o nome e o sobrenome do usuário separadamente.  
Caso algum campo seja deixado em branco, o sistema solicita novamente.

## 🛠️ Tecnologias
- JavaScript

## ▶️ Execução
Executar o arquivo em um navegador.

## 💡 Lógica utilizada
- Estrutura de repetição `while`
- Validação de campos vazios

## ✅ Resultado esperado
Exibir o nome completo corretamente preenchido.
