# Exercício 5 - Saque Bancário

## 📌 Descrição
O programa simula um saque bancário com um saldo inicial.  
O usuário informa valores de saque, e o sistema verifica se há saldo suficiente.  
Caso não haja, solicita um novo valor.  
O programa encerra quando o saldo chega a zero ou negativo.

## 🛠️ Tecnologias
- JavaScript

## ▶️ Execução
Executar o arquivo em um navegador.

## 💡 Lógica utilizada
- Estrutura de repetição `while`
- Condicional `if`
- Controle de saldo

## ✅ Resultado esperado
Realizar saques válidos até zerar o saldo.
